# Модуль интеграции (ПМ.03)

## Описание
Данный проект был разработан в рамках производственной практики.

Реализован модуль для работы с данными боевого оружия персонажей, включая создание, редактирование и хранение информации.

## Функционал
Приложение позволяет добавлять новое оружие, изменять его характеристики и получать список всех объектов.

## Назначение
Модуль может использоваться как часть информационной системы для управления игровыми или тестовыми данными.

## Технологии
Python
