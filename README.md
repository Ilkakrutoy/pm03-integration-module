# Модуль интеграции (ПМ.03)

## Описание
В рамках практики был разработан программный модуль для управления данными боевого оружия персонажей.

Приложение реализует функции создания, редактирования, удаления и хранения информации.

## Архитектура
Проект состоит из нескольких модулей:
- models.py — описание структуры данных
- storage.py — работа с файлом (JSON)
- app.py — логика приложения

## Функционал
- Добавление оружия
- Просмотр списка
- Редактирование
- Удаление
- Сохранение в файл

## Технологии
Python, JSON
